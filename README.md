#Trabajo Práctico: Paradigmas de Programación

## Descripción

Este proyecto fue realizado como parte de una actividad académica con el objetivo de comprender y aplicar distintos paradigmas de programación utilizando Python.

## Lenguaje utilizado

Python

##  Paradigmas implementados

### Programación Orientada a Objetos (POO)

Se desarrolló una clase `CuentaBancaria` que simula operaciones básicas:

* Depósitos
* Retiros
* Consulta de saldo

Este enfoque permite organizar el código mediante objetos que representan entidades del mundo real.

###  Programación Funcional

Se trabajó con funciones como:

 `map()` → transformación de datos
 `filter()` → filtrado de elementos
 `reduce()` → reducción de datos

Este paradigma se centra en el uso de funciones y evita modificar estados directamente.

##  Estructura del proyecto

`poo.py` → Ejemplo de Programación Orientada a Objetos
`funcional.py` → Ejemplo de Programación Funcional

##  Cómo ejecutar

Abrir una terminal en la carpeta del proyecto y ejecutar:

```bash
python poo.py
python funcional.py
```
##  Conclusión

Python permite trabajar con múltiples paradigmas de programación, lo que lo convierte en un lenguaje flexible y potente para distintos tipos de desarrollo.
